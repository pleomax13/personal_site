Отчет об информационном поиске

Представленный отчет позволяет оценить информационную ситуацию по теме магистерской работы. Он является основным документальным подтверждением глубины и полноты информационного поиска, а также служит для фиксации текущей ситуации в исследуемой области.

Поиск выполнен с использованием четырех поисковых систем (Google, Яндекс, Bing, Meta). Результаты сведены в таблицу. Всего произведено 12 запросов, имеющих отношение к магистерской работе. Из них три запроса соответствует названию магистерской работы на трех языках, три запроса с ФИО руководителя, а также шесть запросов с ключевыми понятиями по теме магистерской работы.

Ниже приведены две таблицы с отчетами о поиске, которые разделяет временной промежуток в полтора месяца, а также ряд диаграмм, которые позволяют сравнить основные изменения, произошедшие за этот период.

Анализ результатов

Сравнивая результаты запросов по различным поисковым системам, можно прийти к выводу, ч то наилучшие результаты показала система Google. Во всех случаях было найдено достаточное количество документов на всех языках.

Необходимо отметить, что поисковая система Яндекс при большом числе найденных результатов не дает возможности зафиксировать их точное количество. Поэтому в таблице для Яндекса и приведены приблизительные значения, которые предоставляет поисковая система.

При сравнении и анализе результатов в отчетах о поиске, которые разделяют два месяца, необходимо отметить, что для всех поисковых запросов количество найденных результатов в некотрых случаях выросло, а в некоторых снизилось. Диаграммы сравнения результатов поиска для каждого языка приведены ниже.

Из диаграммы можно сделать вывод, что максимальный рост поисковой выдачи не превышает 100%, а для некоторых запросов количество найденных результатов либо осталось неизменным, либо уменьшилось.

Диаграмма также позволяет оценить изменения для каждого выполненного запроса.

Так, количество найденых результатов в поисковой системе Google по запросу Metrological reliability выросло на 22%, в то же время количество результатов, найденых в поисковой системе Яндекс, упало на 49%.

А по запросу Исследование и разработка методов и средств метрологической надежности оптического адсорбционного шахтного метанометра дела обстоят иначе: в поисковой системе Google количество найденных результатов уменьшилось всего на 1.5%, в то время, как в поисковой системе Яндекс количество результатов возросло на 18%.

В целом, прослеживаются такие тенденции:

  • количество найденных результатов в посковой системе Яндекс практически по всем запросам снизилось;
  • количество найденных результатов в посковой системе Google, в основном, возросло;
  • в поисковых системах Bing и Meta количество найденых результатов, либо осталось неизменными, либо возросло незначительно. Однако, например, по запросу Optical Absorption Shaft Methane Meter количесво результатов возросло на 49% и 100%, соответственно. Случаи, когда на диаграмме зафиксирован рост количества результатов на 100%, по факту количество изменилось лишь на еденицу (при первом поиске было ноль результатов, при втором – один).
